## Overview

This project focused on managing COVID outbreak data, providing real-time analytics and reporting features.

## Key Features

- **Database Management**: Created a structured database for storing patient data and infection rates.
- **Visualization**: Used Matplotlib to generate charts and graphs for trend analysis.
- **GUI Interface**: Built with PySimpleGUI for user-friendly data interaction.

## Technologies Used

- **Python**: For scripting and automation.
- **SQL**: For structured data management.
- **Matplotlib**: For graphical data representation.
- **PySimpleGUI**: For building an interactive interface.
