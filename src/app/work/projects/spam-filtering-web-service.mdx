## Overview

This project focuses on deploying a machine learning-based spam filter as a web service and integrating it with Gmail.

## Key Features

- **Webhooks & API Integration**: Connected with Gmail API for real-time email filtering.
- **Machine Learning Model**: Trained a classifier to identify spam messages.
- **Deployment**: Hosted the spam filtering model as a web service.

## Technologies Used

- **Gmail API**: For email retrieval and filtering.
- **Machine Learning**: Used NLP techniques to classify emails.
- **Python & Flask**: For API implementation and model deployment.
